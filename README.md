Custom URL Shortner

* [Express](https://expressjs.com/)
* [Mongoose](http://mongoosejs.com/)
* [Redis](https://redis.io/)
* [ShortId](https://github.com/dylang/shortid)

## Getting Started

```sh
# clone it
git clone https://github.com/sumitmallick/URLShortner.git
cd URLShortner

# Install dependencies
npm/yarn install

# Start  server
npm/yarn run server
```
